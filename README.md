# CRUD-REACTJS-NODEJS
Study project on ReactJS, NodeJS and MySQL.
Performs the inclusion, editing and deletion of records.

## Change
Change mySQL `server/index.js`

## Installation

### Client

  1. `$ cd client`
  2. `$ npm install`
  3. `$ npm start`
  
### Server

  1. `$ cd server`
  2. `$ npm install`
  3. `$ npm run serverStart`


## Used Libraries

### Front-end

- `@material-ui`
- `Axios`

### Back-end

- `express`
- `mysql`
- `nodemon`
- `cors`
